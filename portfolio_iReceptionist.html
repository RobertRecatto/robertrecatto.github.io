iReceptionist


I worked with a team of 14 to create a receptionist application using using a variety of tools from Angular to Express to Node.

Our application was made for a professor posing as a client who tried to emulate the combination of his most ridiculous clients.

We created interfaces for not only the small businesses using the app but even for the client to manage his buiness in an attempt to imitate the real needs and required features of a modern SaaS application.

App Features


  • - Realtime Updating using sockets

  • - Interactive front end using angular

  • - Stylish interface using bootstrap

  • - MVC Design for easy extendability

  • - Charts and Analytics using bootstrap and JQuery

My Contributions


My main contributions were all on the VIP page, which is meant for the owner of iReceptionist, the person who hired our pretend company to build this application.

I built the pages from complete scratch using a bootstrap theme to help with some of the heavy lifting.

I implemented searches, sorting and queries all using vanilla JS and a little bit of JQuery.

I connected my page to the backend by writing my own services and sending requests to the routes.




What I learned


Some of the most important things I learned from this project were actually not programming based.

I learned how to interact with clients, specifically those with high expectations and little technical knowledge.

However, I also did learn a lot of technical skills as well as it was my first project using Angular and JQuery.
